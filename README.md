# Advent of Code 2025

## Build
```sh
cargo build [--release]
```

## Run

### Run single part
```sh
./target/{debug,release}/aoc2025 [day] [part] < input.txt
```

### Run single day
```sh
./target/{debug,release}/aoc2025 [day] < input.txt
```

### Run all days

Input files will be read from `./input` directory.

```sh
./target/{debug,release}/aoc2025
```
